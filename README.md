# 🎤 تبدیل صدا به متن - Voice to Text Extension

یک افزونه حرفه‌ای برای مرورگر کروم که به شما امکان می‌دهد با استفاده از Google Gemini AI، صدای خود را به متن تبدیل کنید.

## ✨ ویژگی‌ها

- 🎙️ **تبدیل صدا به متن**: تبدیل سریع و دقیق صدا به متن در فیلدهای ورودی وب‌سایت‌ها
- 📝 **فقط برای Input و Textarea**: دکمه‌ها فقط کنار فیلدهای متنی نمایش داده می‌شوند (نه selectbox، password، number و...)
- 🌐 **پشتیبانی از چند زبان**: فارسی و انگلیسی
- 🎨 **رابط کاربری زیبا**: طراحی مدرن و کاربرپسند
- ⏹️ **ویرایش در حین ضبط**: می‌توانید در حین ضبط، تایپ کنید یا Enter بزنید (ضبط خودکار متوقف می‌شود)
- 🎤 **دستورات صوتی هوشمند**: "نقطه" → . | "ویرگول" → ، | "سوال" → ؟ | "خط جدید" → ↵
- ❌ **دکمه حذف هوشمند**: حذف کردن دکمه میکروفن از فیلدهایی که نمی‌خواهید
- 👁️ **قابلیت مخفی/نمایش**: کنترل کلی نمایش دکمه‌های میکروفن در تمام سایت‌ها
- 📄 **فقط Textarea**: نمایش دکمه‌ها فقط کنار textarea (نه input) برای متن‌های بلند
- 🌍 **Whitelist سایت‌ها**: فعال کردن اکستنشن فقط روی سایت‌های خاص
- 🤖 **پشتیبانی از Gemini AI**: استفاده از قدرتمند‌ترین مدل‌های Google (برای API)

## 📦 نصب

1. این پوشه را دانلود کنید
2. مرورگر Chrome را باز کنید
3. به آدرس `chrome://extensions` بروید
4. گزینه "Developer mode" را فعال کنید
5. روی "Load unpacked" کلیک کنید
6. پوشه این افزونه را انتخاب کنید

## 🚀 نحوه استفاده

### مرحله 1: دریافت توکن API

1. به [Google AI Studio](https://aistudio.google.com/app/apikey) بروید
2. توکن API خود را دریافت کنید
3. افزونه را باز کنید و توکن را در قسمت "توکن API" وارد کنید
4. دکمه "ذخیره" را بزنید

### مرحله 2: تنظیمات

- **انتخاب مدل**: مدل Gemini مورد نظر خود را انتخاب کنید (پیشنهاد: gemini-1.5-flash)
- **زبان**: فارسی یا انگلیسی را انتخاب کنید
- **نمایش دکمه**: دکمه‌های میکروفن را فعال یا غیرفعال کنید
- **سایت‌های مجاز** (اختیاری): لیست سایت‌هایی که می‌خواهید اکستنشن روی آنها فعال باشد
  - خالی: همه سایت‌ها فعال
  - پر: فقط سایت‌های لیست
- **نقش سیستم** (اختیاری): دستورات صوتی سفارشی خود را تعریف کنید

### مرحله 3: استفاده از ضبط صوت

1. دکمه‌های میکروفن فقط کنار **input** و **textarea** نمایش داده می‌شوند
2. کنار هر فیلد، دو دکمه خواهید دید:
   - 🎤 دکمه میکروفن (بنفش)
   - ❌ دکمه حذف (خاکستری)
3. روی دکمه 🎤 کلیک کنید تا ضبط شروع شود (قرمز می‌شود)
4. صحبت کنید
5. دوباره روی دکمه کلیک کنید تا ضبط متوقف شود
6. برای حذف دکمه میکروفن، روی دکمه ❌ کلیک کنید
7. برای بازگشت دکمه، صفحه را Refresh کنید

## 🎯 ویژگی‌های پیشرفته

### 1. دستورات صوتی هوشمند

می‌توانید با صدا علامت‌گذاری و فرمت‌دهی انجام دهید:

**دستورات فارسی:**
- "نقطه" → .
- "ویرگول" → ،
- "سوال" → ؟
- "تعجب" → !
- "خط جدید" یا "اینتر" → ↵ (خط بعد)
- "دونقطه" → :
- "نقطه ویرگول" → ;

**دستورات انگلیسی:**
- "period" یا "dot" → .
- "comma" → ,
- "question mark" → ?
- "exclamation" → !
- "new line" یا "enter" → ↵
- "colon" → :
- "semicolon" → ;

**مثال استفاده:**
```
شما می‌گویید: "سلام نقطه چطوری سوال خط جدید من خوبم تعجب"
نتیجه: "سلام. چطوری؟
من خوبم!"
```

فعال/غیرفعال: در popup اکستنشن، بخش "🎙️ دستورات صوتی ساده"

راهنمای کامل: [VOICE_COMMANDS_GUIDE.md](VOICE_COMMANDS_GUIDE.md)

### 2. ویرایش در حین ضبط

قبلاً نمی‌توانستید در حین ضبط تغییری بدهید. حالا:
- ✅ اگر شروع به تایپ کنید → ضبط خودکار متوقف می‌شود
- ✅ اگر Enter بزنید → ضبط خودکار متوقف می‌شود  
- ✅ اگر focus را بردارید → ضبط خودکار متوقف می‌شود

این باعث می‌شود تجربه کاربری روان‌تری داشته باشید!

### 3. فقط برای Textarea

می‌توانید مشخص کنید دکمه‌ها فقط کنار textarea ظاهر شوند، نه input ها:

**حالت عادی (پیش‌فرض):**
- ✅ دکمه کنار input (search box, text field, email, ...)
- ✅ دکمه کنار textarea (نوشتن متن‌های بلند)

**حالت "فقط Textarea":**
- ❌ دکمه کنار input ها ظاهر نمی‌شود
- ✅ دکمه فقط کنار textarea ها (برای متن‌های بلند)

**نحوه فعال‌سازی:**
1. باز کردن popup اکستنشن
2. پیدا کردن "📄 فقط برای Textarea"
3. روشن کردن toggle
4. صفحه خودکار reload می‌شود

**مثال استفاده:**
```
سناریو: فقط در Gmail compose و GitHub issues می‌خواهم voice داشته باشم

راه‌حل:
1. Toggle "فقط Textarea" را روشن کنید
2. Whitelist: gmail.com و github.com
3. حالا فقط در textarea ها دکمه ظاهر می‌شود (UI تمیزتر!)
```

راهنمای کامل: [TEXTAREA_ONLY_TEST.md](TEXTAREA_ONLY_TEST.md)

### 4. Whitelist سایت‌ها

می‌توانید مشخص کنید اکستنشن فقط روی چه سایت‌هایی فعال باشد:

```
google.com          → فقط google.com
*.google.com        → google.com + تمام subdomain ها
youtube.com
github.com
*.wikipedia.org
```

**مثال عملی**:
- برای فعال سازی فقط روی سایت‌های گوگل:
  ```
  *.google.com
  *.youtube.com
  ```

- برای سایت‌های توسعه‌دهندگی:
  ```
  github.com
  stackoverflow.com
  developer.mozilla.org
  ```

راهنمای کامل: [WHITELIST_TEST.md](WHITELIST_TEST.md)

### 4. نقش سیستم (System Role) - فقط برای Gemini API

⚠️ **توجه مهم:** این بخش فقط برای استفاده از Gemini API است، **نه برای تبدیل صدا به متن**.

تبدیل صدا به متن با Web Speech API مرورگر انجام می‌شود (بدون نیاز به Gemini). 
اما اگر می‌خواهید از Gemini برای تولید متن یا پردازش استفاده کنید، می‌توانید System Role تعریف کنید.

**برای دستورات صوتی در تبدیل صدا به متن، از بخش "دستورات صوتی هوشمند" استفاده کنید.**
